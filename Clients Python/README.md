# Utilisation des deux Clients

### Installation des librairies

Le client **REST** nécessite l'**importation** de ces librairie:

 - *Json*
 - *Requests*
 - *Colorama (optionnel)*

Installer avec pip  :

    pip install requests
    pip install colorama

Pour le client **SOAP** les librairies sont :

 - *xml*
 - *lxml*

Installer avec pip :

    pip install xml
    pip install lxml
